# Invoice Maker App (Python Tkinter)

A simple Invoice Generator desktop application built using Python Tkinter.

## Features
- Store settings (name, address, phone, location)
- Customer name and phone input
- Multiple invoice items (name, qty, price)
- Optional GST % and Discount
- Paid via option (Cash/UPI/Card/Bank Transfer)
- Generates PDF invoice using ReportLab
- Saves invoice record in JSON

## Output
- Generates PDF invoice file: INV-xxxx.pdf
- Stores invoices in invoices.json
- Stores settings in settings.json
