# Password Strength Checker (Python)

A simple cybersecurity-focused Python tool that checks the strength of a password.

## Checks Included
- Password length (8+ and 12+)
- Uppercase letters
- Lowercase letters
- Digits
- Symbols

## Output
- Weak ❌
- Medium ⚠️
- Strong ✅

## How to Run
```bash
python main.py
