# To-Do List App (Python Tkinter)

A multi-theme GUI To-Do List application built with Python and Tkinter.

## Features
- Dark Mode / Light Mode / Neon Mode
- Add tasks
- Mark tasks as done
- Delete tasks
- Clear all tasks
- Saves tasks automatically in JSON file

## Data Storage

Tasks are stored in tasks.json
- Tech Used
- Python
- Tkinter
- JSON
